# LiDAR Touch Controller

Проект LiDAR Touch Controller позволяет использовать лидар LD-M1S как сенсорный экран, отслеживая прикосновения к поверхности и управляя курсором мыши на основе полученных данных.

## 📋 Описание

Это приложение предоставляет возможность взаимодействия с компьютером через лидар LD-M1S, который используется в качестве сенсорного устройства. Приложение отслеживает изменения в данных сканирования, соответствующие прикосновению пользователя к поверхности, и преобразует эти данные в действия управления мышью.

### Возможности:
- Калибровка фона помещения
- Фильтрация шума и ложных срабатываний
- Отслеживание точек касания
- Управление курсором мыши (перемещение, клик)
- Калибровка углов рабочей зоны
- Настройка ориентации лидара
- Графический интерфейс для управления всем процессом

## 🔧 Установка

### Требования:
- Python 3.7+
- Windows/Linux/macOS
- Лидар LD-M1S с Ethernet подключением
- Доступ к сети (IP-адрес лидара)

### Шаги установки:

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/artemshaturnyii/Lidar_New/tree/final_ALPHA_TEST
```

2. **Создайте виртуальное окружение (рекомендуется):**
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

4. **Настройте параметры подключения:**
Откройте `config.py` и измените параметры подключения к лидару:
```python
LIDAR_CONFIG = {
    'host': '192.168.0.7',    # IP-адрес вашего лидара
    'port': 25168,            # Порт (по умолчанию для LD-M1S)
    'connection_timeout': 5.0
}
```

## 🖱️ Использование

### 1. Подключение к лидару
При запуске приложение автоматически пытается подключиться к лидару. Статус подключения отображается в панели состояния.

### 2. Создание фоновой карты
Перед началом работы необходимо создать фоновую карту:
- Убедитесь, что перед лидаром нет объектов
- Нажмите кнопку "Create Background Map"
- Приложение соберёт несколько сканов и создаст карту фона

### 3. Профилирование шума (опционально)
Для улучшения точности можно профилировать шум:
- Нажмите "Profile Noise"
- Следуйте инструкциям на экране (убедитесь, что никто не касается поверхности)

### 4. Калибровка углов проекции
Для точного управления мышью необходимо откалибровать углы рабочей зоны:
- Нажмите "Calibrate All Corners" или отдельные кнопки для каждого угла
- Следуйте инструкциям на экране

### 5. Включение управления мышью
После калибровки включите управление мышью:
- Нажмите "Enable Mouse"
- Теперь при касании поверхности курсор будет перемещаться и выполнять клики

### 6. Настройка ориентации
Если лидар установлен не "лицом" к пользователю, можно настроить ориентацию:
- Используйте кнопки "↺ -1°" и "↻ +1°" для точной настройки
- Или установите направление севера через панель "Set North Direction"

## ⚙️ Конфигурация

Основные параметры настраиваются в файле `config.py`:

### Параметры подключения:
```python
LIDAR_CONFIG = {
    'host': '192.168.0.7',      # IP-адрес лидара
    'port': 25168,              # Порт
    'connection_timeout': 5.0   # Таймаут подключения
}
```

### Параметры обработки:
```python
PROCESSING_CONFIG = {
    'background_grid_step': 1.0,    # Шаг сетки фона (градусы)
    'angle_tolerance': 0.5,          # Точность поиска точек (градусы)
    'use_median_filter': True       # Использовать медиану вместо среднего
}
```

### Параметры производительности:
```python
PERFORMANCE_CONFIG = {
    'target_fps': 60,           # Целевой FPS
    'max_touch_points': 5,      # Максимум точек касания
    'scan_point_limit': 50,     # Лимит точек скана
    'update_interval': 0.016,   # Интервал обновления (сек)
    'fast_mode': True           # Быстрый режим обработки
}
```

## 📊 Файлы данных

При работе приложение создаёт следующие файлы:
- `background.npz` - фоновая карта
- `noise_profile.json` - профиль шума
- `persistent_noise_points.json` - постоянный шум
- `projection_corners.json` - углы проекции
- `orientation_config.json` - конфигурация ориентации
- `north_calibration.json` - калибровка севера

Эти файлы автоматически удаляются при завершении работы приложения.

## 🛠️ Разработка

### Структура модулей:

#### calibration
Отвечает за создание и управление фоновой картой:
- `calibrator.py` - основной класс калибровки
- `data_handlers.py` - обработка данных сканов
- `file_io.py` - сохранение/загрузка фоновой карты

#### detection
Модуль обнаружения касаний:
- `touch_detector.py` - основной класс детектора
- `comparators.py` - функции сравнения сканов
- `threshold_config.py` - конфигурация порогов

#### gui
Графический интерфейс:
- `gui_manager.py` - менеджер GUI элементов

#### lidar_sdk
SDK для работы с лидаром:
- `lidar_m1.py` - основной класс лидара
- `connection.py` - управление подключением
- `protocol.py` - протокол обмена данными
- `models.py` - модели данных

#### mouse_controller
Управление курсором мыши:
- `mouse_controller.py` - класс контроллера мыши

#### noise_filter
Фильтрация шума:
- `noise_profiler.py` - профилирование шума
- `noise_filter.py` - фильтр ложных срабатываний
- `corner_calibration.py` - калибровка углов

#### orientation
Управление ориентацией:
- `orientation.py` - менеджер ориентации

#### persistent_noise_filter
Постоянный шум:
- `persistent_noise_manager.py` - менеджер постоянного шума

## 📈 Алгоритмы обработки

### Детекция касаний:
1. Сравнение текущего скана с фоновой картой
2. Вычисление отклонений расстояний
3. Фильтрация по уровню уверенности
4. Группировка точек по угловому окну
5. Применение адаптивных порогов

### Фильтрация шума:
1. Статистическая фильтрация по частоте
2. Отслеживание постоянных точек шума
3. Адаптивная фильтрация по расстоянию

### Управление мышью:
1. Преобразование полярных координат в декартовы
2. Маппинг в координаты экрана
3. Отслеживание касаний для кликов
4. Сглаживание движения курсора
