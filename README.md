# LiDAR Touch Controller

LiDAR Touch Controller - это приложение для управления компьютером с помощью лидара, позволяющее использовать пространство перед устройством как сенсорную поверхность для управления курсором мыши.

## 🎯 Основные возможности

- **Детекция касаний**: Определение приближения руки/пальца к поверхности с помощью LiDAR
- **Управление мышью**: Перемещение и клик курсора на основе обнаруженных касаний
- **Калибровка фона**: Адаптация к окружающей среде и фильтрация постоянных объектов
- **Фильтрация шума**: Устранение ложных срабатываний и постоянного шума
- **Калибровка углов проекции**: Настройка соответствия между зоной действия лидара и экраном
- **Графический интерфейс**: Удобное управление всеми функциями через Tkinter GUI

## 🔧 Установка

### Требования
- Python 3.7+
- LiDAR модель M1 с поддержкой Ethernet
- Доступ к сети (IP-подключение)

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Основные зависимости
- numpy - численные вычисления
- matplotlib - визуализация данных
- Pillow - работа с изображениями
- pyautogui - управление мышью
- tkinter - графический интерфейс (встроен в Python)

## ⚙️ Конфигурация

Основные параметры настраиваются в файле `config.py`:

- Параметры подключения к LiDAR (host, port, timeout)
- Параметры калибровки (количество сканов, шаг сетки)
- Параметры обработки (угловая точность, фильтрация)
- Параметры визуализации (размеры графиков)

## 🚀 Использование

### Запуск приложения
```bash
python app.py
```

### Основные этапы работы

1. **Подключение к LiDAR**
   - Приложение автоматически подключается к LiDAR по указанному IP/порту
   - Статус отображается в интерфейсе

2. **Создание фоновой карты**
   - Нажмите "Create Background Map"
   - Убедитесь, что зона действия лидара свободна
   - Будет собрано 10 сканов для построения эталонной карты

3. **Профилирование шума**
   - Нажмите "Profile Noise"
   - В течение 30 секунд не касайтесь зоны действия
   - Будет создан профиль шума для фильтрации

4. **Калибровка углов проекции**
   - Вариант 1: По отдельным углам ("Top Left", "Top Right", и т.д.)
   - Вариант 2: Все углы сразу ("Calibrate All Corners")
   - Для каждого угла нужно встать в соответствующую точку зоны

5. **Запуск детекции**
   - Нажмите "Start Detection"
   - При приближении руки к зоне будут обнаруживаться касания
   - Курсор мыши будет перемещаться в точку касания

### Управление мышью

- **Enable Mouse** - включить управление мышью
- **Disable Mouse** - выключить управление мышью
- Курсор перемещается в точку первого обнаруженного касания
- Автоматическая фильтрация ложных срабатываний

## 📊 Интерфейс приложения

### Статусы
- **LiDAR**: состояние подключения к лидару
- **Background**: статус фоновой карты
- **Noise Profile**: наличие профиля шума
- **Corners**: статус калибровки углов
- **Mouse Control**: состояние управления мышью

### График
- Полярная диаграмма с текущими данными сканирования
- Фоновая карта отображается синей линией
- Точки касания выделены красными маркерами
- Углы проекции показаны оранжевыми квадратами

### Логирование
- Все события отображаются в нижнем окне лога
- Включает информацию о подключении, калибровке, детекции

## 🛠️ Технические особенности

### Алгоритмы обработки
- **Интерполяция фона**: построение равномерной сетки с медианной фильтрацией
- **Детекция отклонений**: сравнение текущих данных с фоном
- **Адаптивные пороги**: настройка чувствительности в зависимости от расстояния
- **Группировка точек**: объединение близких точек касания
- **Фильтрация шума**: статистический и постоянный фильтры

### Безопасность
- Автоматическое удаление временных файлов при завершении
- Корректное закрытие соединения с LiDAR
- Защита от ложных срабатываний на границах

## 📁 Создаваемые файлы

- `background.npz` - фоновая карта
- `projection_corners.json` - углы проекции
- `noise_profile.json` - профиль шума
- `persistent_noise_points.json` - постоянный шум
