# LiDAR M1 Calibration Tool

Инструмент для калибровки и сбора данных с LiDAR модуля M1 по протоколу PoE.

## 📋 Описание

Это Python-приложение предназначено для:
- Подключения к LiDAR M1 по TCP/IP
- Сбора сканов окружающего пространства  
- Построения фоновой карты (background map)
- Сохранения калибровочных данных в файл

## 🚀 Быстрый старт

### Требования:
- Python 3.7+
- Доступ к LiDAR M1 (подключение Ethernet)

### Установка:

1. **Клонирование репозитория:**
```bash
git clone <repository-url>
cd Lidar_New
```

2. **Создание виртуального окружения (рекомендуется):**
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. **Установка зависимостей:**
```bash
pip install -r requirements.txt
```

4. **Настройка конфигурации:**
Отредактируйте параметры подключения в `config.py`:
- IP адрес LiDAR модуля
- Порт подключения

5. **Запуск:**
```bash
python main.py
```

## ⚙️ Конфигурация

Настройки находятся в файле `config.py`:

### Параметры подключения:
```python
LIDAR_CONFIG = {
    'host': '192.168.0.7',      # IP адрес LiDAR
    'port': 25168,              # Порт подключения
    'connection_timeout': 5.0,  # Таймаут соединения (сек)
}
```

### Параметры калибровки:
```python
CALIBRATION_CONFIG = {
    'default_scan_count': 1,    # Количество сканов для сбора
    'min_scans': 1,             # Минимальное количество
    'max_scans': 100,           # Максимальное количество
}
```

### Параметры обработки:
```python
PROCESSING_CONFIG = {
    'background_grid_step': 1.0,      # Шаг угловой сетки (градусы)
    'angle_tolerance': 0.5,           # Допуск по углу (градусы)
    'use_median_filter': True,        # Использовать медиану вместо среднего
}
```

### Параметры визуализации:
```python
DISPLAY_CONFIG = {
    'enable_plot': True,              # Включить отображение графика
    'plot_size': (8, 8),              # Размер графика
    'plot_title': "Фоновая карта",    # Заголовок графика
}
```

## 📊 Выходные данные

### Файлы:
- `background.npz` - файл с фоновой картой (углы и расстояния)
- Полярный график (если включена визуализация)

### Формат данных:
Фоновая карта представляет собой словарь:
```
{
    угол_в_градусах: расстояние_в_миллиметрах,
    ...
}
```

## 🛠 Архитектура

### Слои приложения:
1. **App Layer** (`main.py`) - координация процесса калибровки
2. **Processing Layer** (`calibration/`) - обработка и анализ данных  
3. **Driver Layer** (`lidar_sdk/`) - низкоуровневое взаимодействие с устройством

### Принципы проектирования:
- **Разделение ответственности** - каждый модуль выполняет одну задачу
- **Чистые функции** - обработка данных без побочных эффектов
- **Модульность** - легко тестируемые и заменяемые компоненты
- **Обработка ошибок** - устойчивость к сбоям связи

## 🔧 Особенности реализации

### Протокол LiDAR M1:
- TCP/IP соединение
- Пакеты фиксированной структуры с CRC8
- Поддержка различных размеров пакетов (12/44 точки)
- Автоматическая синхронизация потока данных

### Алгоритм калибровки:
1. Сбор заданного количества сканов
2. Интерполяция на равномерную угловую сетку
3. Фильтрация выбросов (медианный фильтр)
4. Сохранение результата в файл
